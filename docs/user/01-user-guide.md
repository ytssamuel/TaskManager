## 快速開始

### 1. 註冊帳號

1. 前往登入頁面，點擊「註冊」
2. 填寫以下資訊：
   - 使用者名稱 (3-50 字元)
   - 顯示名稱
   - 電子郵件
   - 密碼 (至少 8 個字元)
3. 點擊「註冊」完成

### 2. 登入系統

1. 輸入電子郵件和密碼
2. 點擊「登入」
3. 成功後自動導向儀表板

### 3. 建立第一個專案

1. 在儀表板點擊「建立專案」按鈕
2. 輸入專案名稱和描述（可選）
3. 點擊「建立」

## 帳號管理

### 註冊新帳號

| 欄位 | 說明 | 驗證規則 |
|------|------|----------|
| 使用者名稱 | 登入用識別名 | 3-50 字元，必須唯一 |
| 顯示名稱 | 頁面顯示名稱 | 可選填 |
| 電子郵件 | 帳號 Email | 有效 email 格式，必須唯一 |
| 密碼 | 登入密碼 | 至少 8 個字元 |

### 登入

- 支援 Email + 密碼登入
- 登入成功後取得 JWT Token，有效期 7 天

### 修改密碼

1. 前往「設定」頁面
2. 輸入當前密碼和新密碼
3. 確認新密碼
4. 點擊「更新密碼」

### 上傳頭像

1. 前往「設定」頁面
2. 點擊頭像區域或「上傳頭像」按鈕
3. 選擇圖片檔案
4. 支援格式：JPG, PNG, GIF
5. 建議尺寸：200x200 像素

### 刪除帳號

1. 前往「設定」頁面
2. 滑到頁面底部
3. 點擊「刪除帳號」
4. 確認刪除（此操作無法復原）

## 專案管理

### 建立專案

1. 點擊「建立專案」按鈕
2. 填寫專案資訊：
   - 專案名稱 (必填)
   - 專案描述 (可選)
3. 點擊「建立」

### 編輯專案

1. 在專案頁面點擊「設定」或編輯按鈕
2. 修改專案名稱或描述
3. 點擊「儲存」

### 刪除專案

1. 在專案設定中點擊「刪除專案」
2. 確認刪除（此操作會刪除所有任務）

### 專案角色

| 角色 | 權限說明 |
|------|----------|
| OWNER | 專案所有者，擁有所有權限 |
| ADMIN | 管理員，可管理成員和任務 |
| MEMBER | 成員，可檢視和操作任務 |

## 看板操作

### 看板列狀態

| 狀態 | 中文名稱 | 說明 |
|------|----------|------|
| BACKLOG | 待整理 | 新任務預設狀態 |
| READY | 準備開始 | 準備開始執行 |
| IN_PROGRESS | 進行中 | 任務正在執行 |
| REVIEW | 待審核 | 待審查完成 |
| DONE | 已完成 | 任務已完成 |

### 拖曳操作

- **移動任務**: 拖曳任務卡片到不同列
- **排序任務**: 在同一列內上下拖曳調整順序
- **列排序**: 拖曳列標題調整列順序

### 列鎖定

- 可將看板列設為「鎖定」狀態
- 鎖定的列，任務必須按順序完成

### 任務優先級

| 優先級 | 中文名稱 | 顏色標識 |
|--------|----------|----------|
| LOW | 低 | 灰色 |
| MEDIUM | 中 | 藍色 |
| HIGH | 高 | 橙色 |
| URGENT | 緊急 | 紅色 |

## 任務管理

### 建立任務

1. 在看板頁面，點擊「新增任務」按鈕
2. 填寫任務資訊：
   - 標題 (必填)
   - 描述 (可選)
   - 狀態 (預設: 待整理)
   - 優先級 (預設: 中)
   - 截止日期 (可選)
3. 點擊「建立」

### 編輯任務

1. 點擊任務卡片開啟詳情
2. 修改任務資訊
3. 自動儲存或點擊「儲存」

### 刪除任務

1. 開啟任務詳情
2. 點擊「刪除」按鈕
3. 確認刪除

### 任務指派

1. 開啟任務詳情
2. 點擊「指派」區塊
3. 選擇專案成員
4. 已指派的成員會顯示頭像

### 任務依賴

1. 開啟任務詳情
2. 點擊「依賴」區塊
3. 選擇前置任務
4. 系統會阻止跳過未完成的依賴任務

### 任務留言

1. 開啟任務詳情
2. 在留言區輸入內容
3. 點擊「發送」
4. 可刪除自己的留言

## 成員邀請

### 邀請新成員

1. 在專案頁面點擊「邀請成員」
2. 輸入成員的電子郵件
3. 選擇角色 (ADMIN 或 MEMBER)
4. 設定邀請連結有效期限
5. 點擊「發送邀請」

### 邀請連結

- 邀請連結格式: `/join/{token}`
- 預設有效期限: 7 天
- 邀請人可取消未接受的邀請

### 接受邀請

1. 點擊邀請連結
2. 系統會驗證邀請狀態
3. 確認後自動加入專案

### 查看邀請狀態

1. 在專案設定中查看「邀請」
2. 可看到所有待處理和已接受的邀請

## API Key 使用

### 建立 API Key

1. 前往「設定」頁面
2. 點擊「API Keys」
3. 點擊「建立新的 API Key」
4. 填寫名稱和選擇權限
5. 點擊「建立」

### API Key 權限

| 權限 | 說明 |
|------|------|
| READ_ONLY | 僅能讀取資料 |
| MEMBER | 可讀寫任務，不能刪除 |
| ADMIN | 可讀寫和管理成員 |
| OWNER | 完整權限 |

### 使用 API Key

在 API 請求中加入 header：
```
X-API-Key: tm_live_xxxxxxxxxxxx
```

### API Key 限制

- 可限定只能訪問特定專案
- 可設定過期時間
- 建立者可隨時刪除

## 常見問題

### Q: 忘記密碼怎麼辦？

目前系統不支援密碼重置功能。請聯繫系統管理員。

### Q: 如何成為專案管理員？

由專案 OWNER 在成員設定中將你升級為 ADMIN。

### Q: 可以刪除已完成的任務嗎？

可以。任務刪除權限：OWNER 和 ADMIN 可刪除任何任務，MEMBER 只能刪除自己建立的任務。

### Q: 為什麼任務無法移動到其他狀態？

可能原因：
1. 該任務有未完成的依賴任務
2. 看板列被鎖定
3. 你沒有該操作的權限

### Q: API Key 有使用限制嗎？

有。開發環境每 15 分鐘最多 1000 次請求，生產環境最多 5000 次。

### Q: 頭像上傳失敗怎麼辦？

請確認：
1. 檔案格式是否為 JPG/PNG/GIF
2. 檔案大小是否超過 5MB
3. 網路連線是否正常
